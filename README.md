# Sparite

> Turn your pixel art into Project Arrhythmia prefabs!

## Features

- Upload pixel art images (PNG, JPEG, etc.)
- Customize prefab settings such as scale, color palette, and more
- Preview generated prefabs before downloading
- Export prefabs compatible with Project Arrhythmia

## Getting Started

Use it [here](https://sparite.reimnop.com)
